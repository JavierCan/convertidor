¿Cómo usar la herramienta?

  1. Carga XML sueltos o archivos ZIP con CFDI.
  2. Elige si necesitas una fila por concepto o por comprobante.
  3. Selecciona las columnas y acomódalas en el orden deseado.
  4. Guarda el layout para usar la misma vista en futuras sesiones.
  5. Genera y descarga el Excel consolidado o el ZIP de archivos individuales.
